# Generación Impacto - Landing Page

Landing page para Generación Impacto, una plataforma de responsabilidad social que conecta estudiantes con altas calificaciones pero dificultades económicas, con estudiantes que necesitan apoyo académico.

## Características

- ✅ Diseño responsive y moderno
- ✅ Tema azul y blanco (propósito educativo)
- ✅ Soporte multiidioma (Español/Inglés)
- ✅ Navegación suave entre secciones
- ✅ Formulario de contacto con validación
- ✅ Secciones: Problema, Solución, Beneficios, Objetivos

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Ejecutar el servidor de desarrollo:
```bash
npm start
```

3. Abrir en el navegador:
```
http://localhost:4200
```

## Construcción

Para construir el proyecto para producción:

```bash
npm run build
```

Los archivos de producción estarán en la carpeta `dist/generacion-impacto-landing`.

## Tecnologías

- Angular 17
- TypeScript
- SCSS
- Reactive Forms
